
React project seed with LiveScript.  Things included:

* LiveScript
* gulp
* bower
* react
* flux
* browserify
* livereload

Usage
-----

Install:

        $ npm i && npm run build

Start development server:

        $ npm run dev

Put your App/Main/Boot code in `app/js/app.ls`.
